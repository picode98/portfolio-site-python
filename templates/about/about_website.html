About This Website

This website is fully custom, as it constitutes an ongoing personal project to learn more about web development (otherwise I would just have created a WordPress instance). It has been rebuilt multiple times using different technology stacks.

The current iteration of this website was built using Python and the Django web framework with the following additional libraries:

  • aiosmtpd, used for testing components of this website which involve a mail server (e.g., the contact form) during development.
  • google-cloud-recaptcha-enterprise , used for ReCAPTCHA validation on the contact form.
  • beautifulsoup4, used for formatting adjustment during page rendering.

The previous iteration of this website was built using Kotlin/Spring Web MVC on Linux (Ubuntu Server).

Libraries used in the current iteration:

  • Kotlin serialization extension (kotlinx.serialization)
  • Spring Boot
    • Starter Web
    • Starter Data JDBC
    • Starter Security
    • Starter Mail
  • Dr. Taro L. Saito's SQLite JDBC
  • Jsoup
  • jQuery
  • Vadikom's jQuery SmartMenus
  • Notify.js

The iteration before that was built on a LAMP (Linux-Apache-MySQL-PHP) stack as a personal project to learn about web development.

Libraries used in the previous iteration:

  • jQuery
  • Ivo Petkov's HTML5DOMDocument
  • Vadikom's jQuery SmartMenus
  • Sass.js (used for SmartMenus theme testing)
  • Notify.js
