Console Calculator version {{ latestVersion.version }}

Want an efficient calculator that can take on advanced calculations?

Features

  • Many built-in functions, including:
    • Trigonometric functions
    • Logarithmic functions
    • Psuedo-random-number generation functions
    • A single-variable bisection-method solver function
    • Numerical integral and derivative functions
  • Table generation
  • Numeric list handling
  • Ability to define custom variables and functions and save them
  • Output options: precision, notation, decimal-fraction
  • Command line option to generate CSV output

Supported Platforms

Currently, only Windows x86 (32-bit) builds are supported; these should run in most Windows environments since Windows 7 (compatibility testing is still underway).

Setup Instructions

For ZIP folders containing builds of this program, run "calculator.exe".

Important: If you receive errors regarding the following missing DLLs:

  • MSVCP140.dll or VCRUNTIME140.dll: Install the Microsoft Visual C++ 2015/2017 Redistributable Package.
  • MSVCP100.dll or MSVCR100.dll: Install the Microsoft Visual C++ 2010 Redistributable Package.
